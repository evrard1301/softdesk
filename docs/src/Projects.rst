Projects
========

Create a Project
----------------

+-----------------+
| POST /projects/ |
+-----------------+

Allows an authenticated user to create a project.

**title**
  The title of the new project.

**description**
  The description of the new project.

**type**
  The type of the new project: BACKEND, FRONTEND, ANDROID or IOS.

**author**
  The author of the new project.
